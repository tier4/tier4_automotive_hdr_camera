# TIERIV AUTOMOTIVE HDR CAMERA ドライバ+ツール社内レポジトリ

## ルール
レジスタ情報等外部に公開できないものが存在しているため、
社内レポジトリと社外レポジトリで分離します。

### ブランチルール
下記ルールにて運用します。

| branch name | 概要  | 
| --- | --- |
| main |  リリースブランチとする.必ず最新のコミットがタグ付けされた状態とし、最新バージョンで安定しているもの。リリースブランチをマージする際はsquash厳禁 |
| release/vx.x.x | 時期リリース準備のブランチとする。PRのマージ先はこちら、Squashの利用可 |
| PR用ブランチ | 必ずブランチ名で対応内容が推定できるようになっていれば、何でも良い.マージしたら削除 |
| その他派生ブランチ |　なるべく作らない | 
### PRルール

- 概要は必ず記載すること
- テストが必要な項目はテスト内容orSlackorJIRAチケットのリンクを書くこと



## ディレクトリ構成

```
.
├── drivers
├── packages
│   └── roscube
├── scripts
├── targets
│   └── roscube
└── tools
```

| directory | 概要 |
| --- | --- |
| drivers | ドライバの実体を置く |
| targets | 各プラットフォーム向けにドライバをビルドできる環境を提供する |
| packages | deb packageの実体を配置 |
| tools | user spaceで動くツール群  |
